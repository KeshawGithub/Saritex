# 🌐 Saritex – Assignment Help Website

Saritex is a modern, responsive **academic assistance website** built using a static web development approach. The platform provides assignment help services, blog resources, and a contact system for students to connect with experts.

---

## 🚀 Live Demo

👉 https://saritex.netlify.app/

---

## 📌 Features

* 🏠 **Home Page** – Clean and professional landing page with service highlights
* 📚 **Services Section** – Showcases academic offerings
* 📝 **Blogs Section** – Informative blog cards for students
* ⭐ **Reviews Section** – Displays user trust indicators
* 📞 **Contact Us Feature** – Users can send messages directly
* 📊 **Google Sheets Integration** – Stores user messages from contact form
* 📱 **Responsive Design** – Works on mobile, tablet, and desktop

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & layout
* **JavaScript** – Interactivity
* **Google Sheets API** – Data storage for contact form

## 📩 Contact Form Integration

The contact form is connected to **Google Sheets**, allowing real-time data collection.

### 🔹 How it works:

1. User fills the contact form
2. Data is sent using JavaScript
3. Stored automatically in Google Sheets

## ⚙️ Setup & Usage

1. Clone the repository:

```bash
git clone https://github.com/KeshawGithub/Saritex.git
```

2. Open the project folder:

```bash

```

3. Run locally:

* Open `index.html` in browser
  OR
* Use Live Server (VS Code recommended)
